АСУ СС Мелстон-сервис

Описание проекта

Автоматизированная система управления сервисной службой предназначена для организации работы с заявками на обслуживание от заказчиков. Система решает следующие задачи:

  • Принимает заявки от клиентов посредством web сайта, email сервера или call центра
  • Назначает ответственных и передает всю необходимую информацию им в мобильное приложение
  • Контролирует получение заявки исполнителем, время выполнения и другие параметры в соответствии с SLA
  • Использует NFC карты MIFARE для подписания электронных актов выполненных работ цифровой подписью
  • Позволяет в реальном времени работать с геопозицией сервисных специалистов (отслеживать перемещения, искать ближайших и т.д.)
  • Формирует различные виды отчетностей для аналитики

Мой вклад

Моей задачей была разработка мобильного приложения и backend API для него. Приложение разрабатывалось на платформе WinRT, а для backend'a был выбран ASP.NET MVC и SQL Server.

Возможности мобильного приложения:
  • Работа со своими заявками и заявками своего региона (получение, поиск, фильтрации, передача другому исполнителю, выполнение и др.)
  • Заявки на плановое и внеплановое обслуживание, заявки на закупку материалов
  • Формирование электронных документов, подписанных цифровой подписью с использованием NFC карт
  • Для поддержки работы в оффлайн режиме был разработан собственный протокол обмена информацией на основе версионности строк
  • Работа с геопозицией, отображение других сотрудников на карте, поиск ближайших машин с необходимыми запчастями и др.
  • Фиксация различной информации с помощью камеры (чеки из магазина, бумажные документы и т.д.)

Биллинговая система для Gett Россия

Описание проекта

Разработанная система интегрируется с системой заказчика и работает с биллингом (данные по поездкам). Основные возможности системы:

  • Предоставляет аналитическую отчетность
  • Формирование документов (счет-фактуры, акты сдачи-приемки, детализации)
  • Рассылка документов клиентам и таксопаркам
  • Удобный интерфейс для проверки и корректировки большого объема данных

Мой вклад

Я реализовал backend на ASP.NET MVC и SQL Server.
